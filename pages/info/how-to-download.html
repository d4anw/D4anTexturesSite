Guide

How To Download

Use the website download buttons, then follow the in-app installer steps.

Step By Step

  1. Go to Home and open the texture pack page you want to install.
  2. Click Download Pack and save the .exe file from the website.
  3. Run the .exe.
  4. Make sure you do not have Stumble Cups opened.
  5. Locate the "Stumble Guys (v0.73)" folder in the .exe tool.
  6. Choose the textures u wanna inject.
  7. Press inject and enjoy.

Windows Security Fix

Some users may see: "These files can't be opened" when they try to run the download.

Cause: Windows marks internet downloads with a hidden "Mark of the Web" (Zone Identifier). On some PCs, security settings block opening those files.

Fix (easiest): Unblock the specific file.

  1. Right-click the downloaded file and open Properties.
  2. At the bottom of the General tab, check Unblock.
  3. Click Apply, then click OK.
  4. Open the file again.

Download Not Working?

Try another browser, disable blocking extensions, and download again.

.exe Not Starting?

Right-click the file and run it as administrator, then try again.
